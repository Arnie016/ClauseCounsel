# ClauseCounsel 🤖⚖️

An AI-powered legal assistant for contract disputes, leveraging OpenAI's GPT models to provide intelligent legal insights and analysis.

## 🌟 Features

### Core Functionality
- 💬 AI-powered legal analysis and responses
- 📊 Interactive visualization of related legal concepts
- ⏱️ Timeline view of case proceedings
- 🔍 Semantic search across conversations
- 💾 Conversation history management

### Technical Features
- Vector similarity search using AstraDB
- Interactive network graphs with PyVis
- Chronological timeline visualization
- Real-time AI responses with OpenAI GPT
- Session state management with Streamlit

## 🚀 Getting Started

### Prerequisites
- Python 3.8 or higher
- Node.js (for certain features)
- OpenAI API key
- AstraDB credentials
